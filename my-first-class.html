Variables in cpp

First topic is variables in C++, now you may be wondering what are variables, well I am here to answer that question. Variables are the like a container. It stores/contains a value, which can be changed later.

for example:

int x = 4;

Here the int is a data-type whereas x is a variable name. here the variable x is assigned the integer value 4.

Operators in C++

Moving on to operators, there are dfferent types of operators in C++. Including arithmetic operators, relational operators, logical operators and bitwise operators. Now, on to the explaination

1. Arithmetic Operators

Arithmetic operators are used to perform mathematical operations. for example:

int x = 4;
int y = 2;
int z = x + y;
int a = x - y;
int b = x * y;
int c = x / y;
Here, we can see that the arithmetic operators are used to perform mathematical operations such as + , - , * , / which refers to plus, minus multiplication and division respectively.

2. Relational Operators

They are used to compare two values. for example:

int x = 4;
int y = 2;
we will be using std::cout which is just used to print something on the terminal/screen.
std::cout << (x == y); //Prints false
std::cout << (x != y); //Prints true
std::cout << (x > y); //Prints true
std::cout << (x < y); //Prints false
from this we can learn about the different types of relational operators including == (equal to), != (not equal to), (>) greater than, (<) less than. the values will be outputted in boolean , i.e. , true or false.
