}

if-else in C++

now we are at one of the first function at coding, the if-else statement. As the name states, there is a condition given, which could either be true or false. If the said condition is true the given block of code will run.
int main(){
int age = 18;
if(age > 18){
std::cout << "You are an adult";
}else{
std::cout << "You are not an adult";
}
return 0;
}
the output on terminal will be:
You are an adult
as the age was 18 the condition was true so the block of code inside if statement was executed. But if the other one was true, the else part would have been used.
